## 💻 | Install

The only dependency is [Jest](https://jestjs.io/es-ES/)

`npm install --save-dev`

To uninstall just delete *node_modules* folder

## 🔧 | Test

Launch all the tests/specific test

`npm test` `npm run test:<num_challenge>`
